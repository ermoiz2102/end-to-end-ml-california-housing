# 🏠 End-to-End Machine Learning Project: California Housing

## 📌 Overview

This project demonstrates a complete machine learning lifecycle from data analysis to deployment. The goal is to predict house prices using multiple regression models and deploy the best-performing model.

---

## 🎯 Objective

* Predict house prices using ML models
* Perform EDA and automated analysis
* Compare multiple algorithms
* Build a reusable ML pipeline
* Deploy model for real-time predictions

---

## 📊 Dataset

* California Housing Dataset (sklearn)
* Contains features like income, rooms, population, and location

---

## 🔍 Exploratory Data Analysis

* Analyzed feature distributions
* Identified correlations
* Used **ydata-profiling** for automated EDA

---

## 🤖 Models Used

* Linear Regression
* Ridge Regression
* Lasso Regression
* Decision Tree Regressor
* Random Forest Regressor

---

## 📈 Model Performance

* Random Forest achieved the best performance
* Evaluated using:

  * MAE
  * MSE
  * RMSE
  * R² Score

---

## 🔄 Pipeline

* Built using sklearn pipeline
* Ensures reusable and scalable workflow

---

## 💾 Model Saving

* Saved using joblib
* Ready for deployment

---

## 🚀 Deployment

* Built web app using Streamlit
* Supports real-time predictions

---

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* ydata-profiling
* Streamlit
